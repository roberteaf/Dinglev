# Blooket Bot (Flood + Crash + Freeze)

A powerful bot for Blooket that can flood games with bots, crash the host, or freeze the leaderboard.

## How to use

1. Install Node.js.
2. Clone this repository.
3. Run `npm install` to install dependencies.
4. Run `node blookflood.js`.
5. Follow the prompts – you'll need a valid Blooket cookie (from a non‑suspended account) and an active game PIN.

## Features

- **Flood**: Join many bots at once (configurable concurrency, optional bypass player limit).
- **Crash**: Send a huge payload to try to crash the host.
- **Freeze Leaderboard**: Send a command to freeze the leaderboard (works in some game modes).
